docker国产管理面板,记录一下国产docker面板安装过程.以及简单使用
发表于 2020-05-03| 更新于 2020-05-05|docker
阅读量:|评论数:

FAST OS DOCKER

更新docker源 (改成阿里源)

拉取镜像并启动

访问 ip:端口 进入面板页面

默认账号密码为 admin:888888

成功进入面板页面

创建一个mysql docker环境

先拉取镜像

拉取完成后，页面会自动刷新，下方的表格内会出现对应的实例id

点击侧边栏中的容器，接着再点击创建容器的按钮

  1. 填写容器名称
  2. 选择容器镜像

    在当前页面的最下方，点击环境变量，添加环境变量MYSQL_ROOT_PASSWORD来设置数据库初始密码

    然后再点击重启策略，选择always，让容器跟docker服务一起热启动

    接着配置端口映射

    最后点击页面中的部署容器来进行创建容器

    创建容器成功后，选择下方表格内对应的容器名，点击上方的开始按钮，启动该容器

    测试一下，能不能连接到容器mysql

    我这里映射的端口是3333

    连接成功!
